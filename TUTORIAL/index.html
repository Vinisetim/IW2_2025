Linguagem Orientada a Objetos


O que é?

Programação Orientada a Objetos é um paradigma de programação que se baseia no conceito de "objetos", que podem conter dados e código: dados na forma de campos (geralmente conhecidos como atributos ou propriedades) e código, na forma de procedimentos (geralmente conhecidos como métodos). Os principais pilares da Linguagem Orientada a Objetos são:

  • Encapsulamento -Ocultar os detalhes internos e mostrar apenas o necessário.
  • Herança -Capacidade de uma classe herdare características de outra
  • Polimorfismo -Capacidade de tratar objetos de maneira "genérica".
  • Abstração -Focar apenas nos aspectos essenciais de um objeto.
